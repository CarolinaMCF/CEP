Sobre HTML

O elemento HTML <title< (Elemento HTML Título) define o título do documento, mostrado na barra de título de um navegador ou na aba da página.
Exemplo: <title<Título incrível<title<


Meta tags são elementos (comandos em HTML) que você cria para descrever o conteúdo de uma página para os mecanismos de busca.
Exemplo: <meta name="viewport" content="width=device-width, initial-scale=1.0"


O elemento HTML <head < providencia informações gerais (metadados) sobre o documento, incluindo seu título e links para scripts e folhas de estilos.
Exemplo:<head<
<meta charset="UTF-8"<
<meta http-equiv="X-UA-Compatible" content="IE=edge"<
<meta name="viewport" content="width=device-width, initial-scale=1.0"<
<title>Colégio Estadual do Paraná<title<
<head<


O elemento HTML Div define uma divisão ou seção em um documento HTML.
Exemplo:<div class="estudante-div"><

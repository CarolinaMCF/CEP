Interruptor do Led

Material utilizado:
- Jumpers
- Arduino
- Protoboard
- Push button
- Resistores


Conceitos importantes: Funcionamento do push button.


Montagem esquemática:
1- Conectar um jumper no GND do arduino ao segmento negativo do protoboard.
2- Conectar um jumper na porta 13 do arduino ao conector I2 do protoboard.
3- Conectar um jumper na porta 12 do arduino ao conector D10 do protoboard.
4- Conectar um led nos conectores F2 (ânodo) e F3 (cátodo).
5- Conectar um jumper no outro GND do arduino ao segmento negativo inferior do protoboard.
6- Conectar um jumper na porta 5V do arduino ao segmento positivo do protoboard.
7- Conectar um push button nos conectores E10, E12, F10 e F12 do protoboard.
8- Conectar um jumper no segmento negativo ao conector A12 do protoboard.
9- Conectar um resistor de 220 ohms no segmento negativo e no conector J3 do protoboard.
10- Conectar um resistor de 10000 ohms no segmento negativo inferior e no conector B10 do protoboard.


Programação:
int botao = 0;
void setup()
{
pinMode(13, OUTPUT);
pinMode(12, OUTPUT);
pinMode(12, INPUT);
digitalWrite(13, LOW);
digitalWrite(12, LOW);
}
void loop()
{
botao = digitalRead(12);
if (botao == HIGH) {
digitalWrite(13, HIGH);
}
digitalWrite(13, LOW);
delay(10);
}


Relato de observação: O led acende ao clicarmos no push button.

3- Conectar um jumper no segmento negativo ao conector H10 do protoboard.
4- Conectar um Buzzer aos conectores J10 (positivo) e J15 (negativo) do protoboard.

1º Projeto: Semáforo

Material utilizado:
- Jumpers
- Arduino
- Protoboard
- Buzzer
- Push button
- Leds
- Resistores


Conceitos importantes:
- Buzzer - transmitir energia em som.
- Funcionamento do push button com os leds.


Algoritmo de montagem:
1- Conecte um jumper no GND do Arduino ao segmento negativo superior do Protoboard.
2- Conecte um jumper na porta 5V do Arduino ao segmento positivo inferior do Protoboard.
3- Para o semáforo de carros, conecte os leds das seguintes cores nos respectivos conectores do protoboard: um led vermelho nos conectores H3 (cátodo) e H4 (ânodo); um led amarelo nos conectores H7 (cátodo) e H8 (ânodo) e um led verde nos conectores H11 (cátodo) e H12 (ânodo).
4- Para o semáforo de pedestres, conecte um led vermelho aos conectores H26 (cátodo) e H27 (ânodo) e um led verde aos conectores H29 (cátodo) e H30 (ânodo).
5- Conecte 5 resistores de 100 ohms aos respectivos conectores, ligando os dois lados do protoboard: F4 (terminal 2) e D4 (terminal 1); F8 (terminal 2) e D8 (terminal 1); F12 (terminal 2) e D12 (terminal 1); F27 (terminal 2) e D27 (terminal 1); F30 (terminal 2) e D30 (terminal 1).
6- Conecte um Push Button, ligando os dois lados do Protoboard, aos respectivos conectores: F22 (terminal 1b), F24 (terminal 2b), E22 (terminal 1a) e E24 (terminal 2a).
7- Conecte o terminal 2 de um resistor de 10.000 ohms ao segmento negativo superior do Protoboard e o terminal 1 ao conector J22.
8- Posicione um Buzzer acima do Protoboard e, utilizando um jumper, conecte seu lado negativo ao segmento negativo da placa.
9- Conecte um resistor de 200 ohms, ligando os dois lados do Protoboard, aos conectores F15 (terminal 2) e D15 (terminal 1).
10- Conecte um jumper ao lado positivo do Buzzer e ao conector I15.
11- Conecte 5 jumpers, ligando o segmento negativo superior do Protoboard aos respectivos conectores: 1º jumper – G3; 2º jumper – G7; 3º jumper – G11; 4º jumper – G26; 5º jumper – G29.
12- Com um jumper vermelho, conecte a porta 13 do Arduino ao conector A4 do Protoboard.
13- Com um jumper amarelo, conecte a porta 12 do Arduino ao conector A8 do Protoboard.
14- Com um jumper verde, conecte a porta 11 do Arduino ao conector A12 do Protoboard.
15- Com um jumper turquesa, conecte a porta 10 do Arduino ao conector A15 do Protoboard.
16- Com um jumper rosa, conecte a porta 8 do Arduino ao conector A22 do Protoboard.
17- Com um jumper preto, conecte o segmento positivo inferior do Protoboard ao conector A24.
18- Com um jumper vermelho, conecte a porta 7 do Arduino ao conector A27 do Protoboard.
19- Com um jumper verde, conecte a porta 6 do Arduino ao conector A30 do Protoboard, finalizando a montagem do Semáforo.


Programação:
// Definição das variáveis
int botao=8;
int carrovermelho=13;
int carroamarelo=12;
int carroverde=11;
int pedestrevermelho=7;
int pedestreverde=6;
int som=10;
int start=0;
// Definir se a variável é de entrada (INPUT) ou saída (OUTPUT)
void setup(){
pinMode(botao, INPUT);
pinMode(carrovermelho, OUTPUT);
pinMode(carroamarelo, OUTPUT);
pinMode(carroverde, OUTPUT);
pinMode(pedestrevermelho, OUTPUT);
pinMode(pedestreverde, OUTPUT);
pinMode(som, OUTPUT);
}
void loop(){
digitalWrite(carroverde, HIGH); // Sinal verde dos carros está ligado.
digitalWrite(pedestrevermelho, HIGH); // Sinal vermelho dos pedestres está desligado.
start= digitalRead(botao);
//Se o botão for acionado, o sinal verde dos carros desliga e o sinal amarelo liga.
if (start == HIGH){
digitalWrite(carroverde, LOW);
digitalWrite(carroamarelo, HIGH);
delay(1000); // 1 segundo de espera.
digitalWrite(carroamarelo, LOW); // Sinal amarelo desliga.
digitalWrite(carrovermelho, HIGH); // Sinal vermelho dos carros liga.
delay(2000); // 2 segundos de espera.
for (int y=0; y<10; y++){
digitalWrite(pedestrevermelho, LOW); // Sinal vermelho do pedestre desliga.
digitalWrite(pedestreverde, HIGH); // Sinal verde do pedestre liga.
tone(som, 440, 500); // O Buzzer emite som.
delay(1000); // 1 segundo de espera.
}
for (int x=0; x<5; x++){
digitalWrite(pedestreverde, LOW); // Sinal verde do pedestre desliga.
tone(som, 440, 500); // Buzzer emite som.
digitalWrite(pedestrevermelho, HIGH); // Sinal vermelho do pedestre liga.
delay(500); // 0,5 segundos de espera.
digitalWrite(pedestrevermelho, LOW); // Sinal vermelho do pedestre desliga.
delay(500); // 0,5 segundos de espera.
}
digitalWrite(pedestrevermelho, HIGH); // Sinal vermelho do pedestre liga.
delay(3000); // 3 segundos de espera.
digitalWrite(carrovermelho, LOW); // Sinal vermelho dos carros desliga.
digitalWrite(carroverde, HIGH); // Sinal verde dos carros liga.
delay(15000); // O botão só poderá ser pressionado novamente após 15 segundos.
}
}


Relato de observação: Ao clicarmos no botão, o sinal do pedestre é acionado e o Buzzer emite som.

Led RGB

Material utilizado:
- Jumpers macho-macho
- Arduino Uno
- Protoboard
- Potenciômetro
- Resistor de 220 ohms
- Led RGB


Conceitos importantes:
- O Led RGB possibilita a emissão de um amplo espectro de luz visível através das cores primárias vermelho, verde e azul (red, green, blue).
- Funcionamento do potenciômetro com o Led RGB.


Algoritmo de montagem:
1- Posicionamos o Led RGB e o Potenciômetro no Protoboard.
2- Conectamos um dos terminais do resistor no terminal negativo do LED RGB, e o outro terminal na parte inferior do Protoboard.
3- Conectamos um jumper ao GND do Arduino e ao segmento negativo do Protoboard.
4- Conectamos um jumper ao 5V do Arduino e ao segmento positivo do Protoboard.
5- Conectamos um jumper no primeiro terminal do Potenciômetro ao segmento positivo do Protoboard.
6- Conectamos um jumper no último terminal do Potenciômetro ao segmento negativo do Protoboard.
7- Conectamos um jumper no terminal central do Potenciômetro à porta A0 do Arduino.
8- Conectamos 3 jumpers às portas 2, 3 e 4 do Arduino ao Led RGB.
9- Conectamos um jumper do resistor ao segmento positivo do Protoboard.


Programação:
// Define o pino analógico A0 para o potenciômetro. //
int Pin_pot = A0;
// Variável que armazenará os dados do potenciômetro. //
int Pot = 0;
// Define o pino digital 2 para o LED Vermelho. //
int Led_R = 2;
// Define o pino digital 3 para o LED Verde. //
int Led_G = 3;
// Define o pino digital 4 para o LED Azul. //
int Led_B = 4;
// Variáveis para controle dos LEDs. //
int ligar;
int desligar;
void setup() {
// Define o pino Led_R como SAÍDA. //
pinMode(Led_R, OUTPUT);
// Define o pino Led_G como SAÍDA. //
pinMode(Led_G, OUTPUT);
// Define o pino Led_B como SAÍDA. //
pinMode(Led_B, OUTPUT);
// Rotina que verifica o tipo de LED RGB utilizado e //
// Adequa os parâmetros para ligar e desligar o LED. //
if (modelo[0] != ‘C’) {
ligar = 0;
desligar = 1;
} else {
ligar = 1;
desligar = 0;
}
// Inicia com os LEDs desligados. //
digitalWrite(Led_R, desligar);
digitalWrite(Led_G, desligar);
digitalWrite(Led_B, desligar);
}
void loop() {
// Armazena os dados do potenciômetro na variável Pot. //
Pot = analogRead(Pin_pot);
// Se Pot estiver entre 0 e 341, faça... //
if (Pot >= 0 && Pot <= 341) {
// Ligue o LED Vermelho. //
digitalWrite(Led_R, ligar);
// Senão... //
} else {
// Desligue o LED Vermelho. //
digitalWrite(Led_R, desligar);
}
// Se Pot estiver entre 342 e 682, faça... //
if (Pot >= 342 && Pot <= 682) {
// Ligue o LED Verde. //
digitalWrite(Led_G, ligar);
// Senão... //
} else {
// Desligue o LED Verde. //
digitalWrite(Led_G, desligar);
}
// Se Pot estiver entre 686 e 1023, faça... //
if (Pot >= 683 && Pot <= 1023) {
// Ligue o LED Azul. //
digitalWrite(Led_B, ligar);
// Senão... //
} else {
// Desligue o LED Azul. //
digitalWrite(Led_B, desligar);
}
}

2º Projeto: Central de Segurança

Material utilizado:
- Jumpers
- Arduino Uno
- Protoboard
- 4 Push buttons
- 1 Led verde
- 1 Led vermelho
- Sensor LDR
- Resistores: 2 de 10.000 ohms e 2 de 220 ohms
- Buzzer


Conceitos importantes:
- Funcionamento do sensor LDR, dos botões, do Buzzer e dos leds.


Montagem esquemática:
1- Conecte um jumper no GND do Arduino ao segmento negativo inferior do Protoboard.
2- Conecte um jumper na porta 5V do Arduino ao segmento positivo inferior do Protoboard.
3- Conecte um jumper do segmento negativo inferior do Protoboard ao segmento negativo superior do Protoboard.
4- Conecte um jumper do segmento positivo inferior do Protoboard ao segmento positivo superior do Protoboard.
5- Conecte quatro Push Buttons, ligando os dois lados do Protoboard, aos respectivos conectores:
- 1º botão - F2 (terminal 1b), F4 (terminal 2b), E2 (terminal 1a) e E4 (terminal 2a);
- 2º botão – F5 (terminal 1b), F7 (terminal 2b), E5 (terminal 1a) e E7 (terminal 2a);
- 3º botão – F8 (terminal 1b), F10 (terminal 2b), E8 (terminal 1a) e E10 (terminal 2a);
- 4º botão – F13 (terminal 1b), F15 (terminal 2b), E13 (terminal 1a) e E15 (terminal 2a).
6- Conecte 4 jumpers, ligando o segmento negativo superior do Protoboard aos respectivos conectores: 1º jumper – J2; 2º jumper – J5; 3º jumper – J8; 4º jumper – J13.
7- Conecte um jumper da porta 13 do Arduino ao conector D4 do Protoboard.
8- Conecte um jumper da porta 12 do Arduino ao conector D7 do Protoboard.
9- Conecte um jumper da porta 11 do Arduino ao conector D10 do Protoboard.
10- Conecte um jumper da porta 10 do Arduino ao conector D15 do Protoboard.
11- Posicione um Buzzer acima do Protoboard e, utilizando um jumper, conecte seu lado negativo ao segmento negativo da placa.
12- Conecte um jumper ao lado positivo do Buzzer e ao conector G17 do Protoboard.
13- Conecte um resistor de 10.000 ohms, ligando os dois lados do Protoboard, aos conectores F17 (terminal 2) e D17 (terminal 1).
14- Conecte um jumper da porta 8 do Arduino ao conector C17 do Protoboard.
15- Conecte o sensor LDR aos conectores G22 (terminal 1) e H22 (terminal 2).
16- Conecte um jumper da porta A0 do Arduino ao conector I22 do Protoboard.
17- Conecte um resistor de 10.000 ohms, ligando os dois lados do Protoboard, aos conectores F22 (terminal 2) e D22 (terminal 1).
18- Conecte um jumper do segmento negativo do Protoboard ao conector C22.
19- Conecte um led verde aos conectores H25 (cátodo) e H26 (ânodo).
20- Conecte um led vermelho aos conectores H28 (cátodo) e H29 (ânodo).
21- Conecte um jumper do segmento negativo do Protoboard ao conector I25.
22- Conecte um jumper do segmento negativo do Protoboard ao conector I28.
23- Conecte um resistor de 220 ohms, ligando os dois lados do Protoboard, aos conectores G26 (terminal 2) e E26 (terminal 1).
24- Conecte um resistor de 220 ohms, ligando os dois lados do Protoboard, aos conectores G29 (terminal 2) e E29 (terminal 1).
25- Conecte um jumper da porta 7 do Arduino ao conector D26 do Protoboard.
26- Conecte um jumper da porta 6 do Arduino ao conector D29 do Protoboard, finalizando a montagem da Central de Segurança.


Programação:
String senha = "123123"; // Definição de senha
String digitado = "";
int t = 0;
int buzzerPin = 8; // Pino do buzzer
int ldrPin = A0; // Pino do sensor LDR
int validationButtonPin = 10; // Pino do botão de validação
int greenLedPin = 7; // Pino do LED verde
int redLedPin = 6; // Pino do LED vermelho
bool alarmArmed = false;
void setup() {
pinMode(7, OUTPUT);
pinMode(6, OUTPUT);
pinMode(13, INPUT_PULLUP);
pinMode(12, INPUT_PULLUP);
pinMode(11, INPUT_PULLUP);
pinMode(buzzerPin, OUTPUT);
pinMode(ldrPin, INPUT);
pinMode(validationButtonPin, INPUT_PULLUP);
pinMode(greenLedPin, OUTPUT);
pinMode(redLedPin, OUTPUT);
Serial.begin(9600);
}
void loop() {
int lightValue = analogRead(ldrPin);
if (!alarmArmed && lightValue < 100) {
armAlarm();
}
if (alarmArmed) {
checkButtons();
}
if (digitalRead(validationButtonPin) == LOW && alarmArmed) {
validatePassword();
}
}
void armAlarm() {
digitalWrite(greenLedPin, LOW);
digitalWrite(redLedPin, HIGH);
alarmArmed = true;
t = 0;
digitado = "";
noTone(buzzerPin); // Desliga o buzzer
}
void checkButtons() {
if (t < senha.length()) {
if (digitalRead(13) == LOW) {digitado += "1"; delay(100); t++;}
if (digitalRead(12) == LOW) {digitado += "2"; delay(100); t++;}
if (digitalRead(11) == LOW) {digitado += "3"; delay(100); t++;}
}
}
void validatePassword() {
if (t == senha.length()) {
if (digitado == senha) {
digitalWrite(greenLedPin, HIGH);
digitalWrite(redLedPin, LOW); // Desliga o LED vermelho
} else {
digitalWrite(redLedPin, HIGH);
playIncorrectSound();
}
t = 0;
digitado = "";
}
}
void playIncorrectSound() {
tone(buzzerPin, 1000);
delay(100);
noTone(buzzerPin); // Desliga o buzzer
}

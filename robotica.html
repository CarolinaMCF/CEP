Interruptor do Led

Material utilizado:
- Jumpers
- Arduino
- Protoboard
- Push button
- Resistores


Conceitos importantes: Funcionamento do push button.


Montagem esquemática:
1- Conectar um jumper no GND do arduino ao segmento negativo do protoboard.
2- Conectar um jumper na porta 13 do arduino ao conector I2 do protoboard.
3- Conectar um jumper na porta 12 do arduino ao conector D10 do protoboard.
4- Conectar um led nos conectores F2 (ânodo) e F3 (cátodo).
5- Conectar um jumper no outro GND do arduino ao segmento negativo inferior do protoboard.
6- Conectar um jumper na porta 5V do arduino ao segmento positivo do protoboard.
7- Conectar um push button nos conectores E10, E12, F10 e F12 do protoboard.
8- Conectar um jumper no segmento negativo ao conector A12 do protoboard.
9- Conectar um resistor de 220 ohms no segmento negativo e no conector J3 do protoboard.
10- Conectar um resistor de 10000 ohms no segmento negativo inferior e no conector B10 do protoboard.


Programação:
int botao = 0;
void setup()
{
pinMode(13, OUTPUT);
pinMode(12, OUTPUT);
pinMode(12, INPUT);
digitalWrite(13, LOW);
digitalWrite(12, LOW);
}
void loop()
{
botao = digitalRead(12);
if (botao == HIGH) {
digitalWrite(13, HIGH);
}
digitalWrite(13, LOW);
delay(10);
}


Relato de observação: O led acende ao clicarmos no push button.

Buzzer

Material utilizado:
- Jumpers
- Arduino
- Protoboard
- Buzzer


Conceitos importantes:
- Buzzer - transmitir energia em som.
- Porta PWM - ~ 3, 5, 6, 9, 10, 11
x++
x = x+1
x--
x = x-1


Montagem esquemática:
1- Conectar um jumper no GND do arduino ao segmento negativo do protoboard.
2- Conectar um jumper na porta 8 do arduino ao conector F15 do protoboard.
3- Conectar um jumper no segmento negativo ao conector H10 do protoboard.
4- Conectar um Buzzer aos conectores J10 (positivo) e J15 (negativo) do protoboard.


Programação:
#define buzzer 8
void setup()
{
pinMode(buzzer, OUTPUT);
noTone(buzzer);
}
void loop()
{
tone(buzzer, 262, 1000);
tone(buzzer, 294, 1000);
tone(buzzer, 330, 1000);
tone(buzzer, 349, 1000);
tone(buzzer, 392, 1000);
}

Relato de observação: Após montar, emitiu som do Buzzer.
